# Evidencia

Aquí va todo lo que **prueba** que el proyecto funciona y que hay demanda. Súbelo antes del tag `v1.0-final`.

## 1. Screenshots (crea la carpeta `screenshots/`)
Capturas recomendadas:
- [ ] Landing completa (o varias secciones).
- [ ] Canvas de n8n del flujo **rutina inicial** (con todos los nodos).
- [ ] Canvas de n8n del flujo **seguimiento**.
- [ ] Canvas del **Error Trigger** + el ajuste "Error Workflow" en Settings.
- [ ] El ajuste **Retry On Fail** y **On Error: Continue** en un nodo (para probar los 3 mecanismos).
- [ ] Google Sheet con las pestañas `general`, `seguimiento`, `hojas_clientes`.
- [ ] Una **hoja de cliente** creada automáticamente, con ejercicios.
- [ ] Los **correos** recibidos (rutina inicial y rutina ajustada).
- [ ] La pestaña **Executions** de n8n mostrando ejecuciones exitosas.

## 2. Video de demo

7. Cierra explicando el loop: datos reales → IA ajusta (10 seg).
